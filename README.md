# Scott County Pool Scheduler

## Overview
The Scott County Pool Scheduler is a Python-based workforce scheduling system designed for Scott County Parks & Recreation. It automates scheduling for over 100 seasonal employees while enforcing staffing rules, employee certifications, availability, overtime limits, and scheduling restrictions.

## Features
- Automatic monthly schedule generation
- Employee availability management
- Vacation and day-off requests
- Weekly overtime prevention
- Overlapping shift detection
- Headguard and Lifeguard certification validation
- Pool-specific staffing requirements
- Excel import and export
- Employee hours reports
- Unfilled shift reporting

## Technologies
- Python
- OpenPyXL
- Excel
- CSV
- Scheduling Algorithms
- Workforce Management

## Outputs
The scheduler generates:
- Monthly Schedule
- Employee Hours Report
- Employee Schedule
- Unfilled Shifts Report
